# AWS EMR Spark Analytics Pipeline

## Overview
Batch analytics pipeline built on AWS using EMR and Spark
to process large datasets stored in Amazon S3.

## Architecture
S3 (Raw) → Spark (EMR) → S3 (Processed)
→ Glue Data Catalog → Athena / Redshift

## Technologies
- AWS EMR
- Apache Spark
- Amazon S3
- AWS Glue
- Athena / Redshift

## Use Cases
- Batch analytics
- Data warehousing
- Large-scale transformations
